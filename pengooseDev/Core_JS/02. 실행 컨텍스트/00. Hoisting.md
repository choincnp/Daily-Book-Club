## 호이스팅

> 변수와 함수의 메모리 공간을 선언 전에 미리 할당하는 것.

변수, 함수

```js
function foo1() {} // 선언식
const foo2 = function () {}; // 표현식 const에 할당되었기 때문에 Hoisting X
const foo3 = () => {}; // Arrow Func const에 할당되었기 때문에 Hoisting X
```
